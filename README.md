# Portfolio Website

This is a personal portfolio website built with Spring Boot.

## Features
- Home, About, Projects, and Contact pages
- Simple contact form (no email sending by default)
- Modern, clean design using HTML, CSS, and Thymeleaf

## Getting Started

### Prerequisites
- Java 17 or higher
- Maven

### Running the Project

1. Open a terminal in the project root directory.
2. Run:
   mvn spring-boot:run
3. Visit http://localhost:8080 in your browser.

---

You can customize the content in the templates and static folders.
